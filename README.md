## TL; DR

在這篇文章中，我們接續「Jupyter 新增選單的 Terminal 是什麼？」介紹其他能夠操作檔案與資料夾的基礎指令，並提供環境鼓勵讀者可以實際操作。

1. `cd`: 切換到某個資料夾
2. `cp`: 複製檔案和資料夾
3. `mv`: 移動檔案和資料夾
4. `rm`: 刪除檔案或資料夾
